## 容易遺漏的小東西

### 階段 1（會走的車）

- [ ] 🔋 **電池本體** — 套件只給空的電池盒（常見 18650 或 3~4 顆 3 號電池）。**先確認盒子吃哪種**再買。
  - **可用充電電池（推薦）**：18650 本身就是充電鋰電，3 顆串聯 ≈ 11.1V，電壓足最穩（要配鋰電充電器）。
  - AA 盒可用充電 NiMH，但 6 顆只 7.2V（鹼性是 9V），**較沒力、電快沒時 Uno 可能不穩**。→ 能選 18650 就選 18650。
- [ ] 🔌 **Arduino USB 線** — 寫程式進板子用（看接頭 type-B / micro / type-C）
- [ ] 🪛 小螺絲起子 — 組裝底盤（多數套件附，沒附要自備）

### 階段 3（樹莓派上車跑 YOLO）

- [x] 💾 **microSD 記憶卡** — 已插在 Pi 上、**裡面有可開機上網的系統** ✅（IP `192.168.169.91` 連得通＝系統已就緒，省下灌系統這步）
- [ ] 🔋 **行動電源**（給樹莓派）— 車在跑時獨立供電；**馬達和樹莓派要分開供電**，否則馬達啟動時電壓掉、樹莓派會當機重開
- [ ] 🔌 樹莓派桌上測試電源 — 5V 電源供應器（手機充電頭多半可用）

## 套件決定

**選定：傑森創工 Uno 2WD 自走車套件**（評比多款後）。

- **為什麼這款**：台灣老牌賣家，附**完整中文教學 + 接線圖 + 範例程式**，循序漸進（組裝 → L298N 入門 → 避障 → 循線 → 藍牙）。自學最需要的就是這個，其他款只有陽春教學卡。
- **內容**：Uno + 感測器擴展板 + L298N + SR04 超音波 + 藍牙(HC-05/06) + 紅外循跡 + 底盤/馬達/輪 + 杜邦線/螺絲/束帶。
- **功能**：超音波避障、紅外循跡、藍牙手機遙控 → 階段 1「遙控車」直接達成。
- **賣家警告「需 Arduino 基礎、不幫查接線」不影響**：使用者是工程師 + 有接線圖範例 + 有 Claude 協助 debug。

### 為什麼這台用 Uno 而不用 Mega

- 完整小車套件（底盤、shield、教學）全是為 **Uno 尺寸**設計；Mega 較大，shield 插不上、鎖孔不合。
- → **這台車用套件附的 Uno；MEGA 2560 留作備用 / 未來較大的專案。** Mega 不浪費。

## Arduino 連線設定（排雷記錄 2026-06-04）

- 板子：**Arduino MEGA 2560**（相容板 clone，USB 晶片是 **CH340**，VID_1A86 PID_7523）
- **第一坑：USB 線**——舊線是「只能充電」的，板子不亮也不被偵測。換成可傳資料的線才行。
- **第二坑：驅動**——clone 板要裝 **CH340 驅動**（WCH 官網 CH341SER.EXE）。
- **第三坑：Code 10**——裝完驅動仍報「裝置無法啟動 (Code 10)」。解法：裝置管理員 →
  連接埠 → USB-SERIAL CH340 → **停用裝置 → 啟用裝置**，即恢復正常（免重開機）。
- 結果：目前掛在 **COM5**，狀態 OK。（COM 編號可能隨插拔/換埠改變）
- Arduino IDE 設定：開發板 **Arduino Mega or Mega 2560** ／ 處理器 **ATmega2560** ／ 序列埠 **COM5**
