## 兩種「眼睛」別搞混

| 長相 | 真實身份 | 能做什麼 | 誰用 |
|---|---|---|---|
| 兩個圓銀色小圓筒 | 超音波感測器（HC-SR04） | 只測「前方幾公分有東西」，分不出草或石頭 | Arduino（小腦），近距離避障 |
| 一顆有鏡片的鏡頭 | 真‧鏡頭 / webcam | 拍影像，YOLO 才能辨識 | 樹莓派（大腦），視覺辨識 |

自走車套件附的「眼睛」幾乎都是**超音波感測器**（不是 YOLO 要的鏡頭）。
→ 待確認：打開套件看那個眼睛是「兩個圓銀筒」（超音波）還是「鏡片鏡頭」。

## 容易遺漏的小東西

### 階段 1（會走的車）

- [ ] 🔋 **電池本體** — 套件只給空的電池盒（常見 18650 或 3~4 顆 3 號電池；18650 還要充電器）
- [ ] 🔌 **Arduino USB 線** — 寫程式進板子用（看接頭 type-B / micro / type-C）
- [ ] 🪛 小螺絲起子 — 組裝底盤（多數套件附，沒附要自備）

### 階段 3（樹莓派上車跑 YOLO）

- [ ] 💾 **microSD 記憶卡** — 樹莓派的「硬碟」，要灌作業系統，建議 32GB 以上（沒讀卡機就加一個）
- [ ] 🔋 **行動電源**（給樹莓派）— 車在跑時獨立供電；**馬達和樹莓派要分開供電**，否則馬達啟動時電壓掉、樹莓派會當機重開
- [ ] 🔌 樹莓派桌上測試電源 — 5V 電源供應器（手機充電頭多半可用）

## 套件提醒（避免買錯）

- 注意「**含不含主板**」：很多套件附 Arduino Uno。已有 Arduino 可買「不含主板」版省錢；含主板也無妨（多一塊備用）。
- 套件通常一次包含：底盤 + 2 顆馬達 + 輪子 + 馬達驅動板（L298N）+ 電池盒 + 杜邦線。買前掃一眼清單有沒有這幾樣。
