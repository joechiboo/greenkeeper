從網頁模擬器，到一台真的會自己走的機器

← 回模擬器 GitHub

這是什麼

GreenKeeper 一開始是個網頁模擬器，比較除草機用不同走法走遍院子的效率。

下一步的目標：做出一台真實、會自己走的機器。模擬器不會白做——它是這台機器的「大腦練習場」，演算法在模擬器調好，真機只要把兩端的「假器官」換成真的。

🎈 這是個好玩、自學的專案，沒有時程壓力。下面的階梯是「地圖」不是「死線」——有空、有零件、有心情，再往上爬一階就好。

一台機器的三層

👁
感知 · 眼睛
鏡頭 + YOLO 看出「草 / 障礙物」
🧠
決策 · 大腦
決定往哪走（= 模擬器在練的）
🦿
動作 · 手腳
左右輪轉速不同 → 車就轉彎

四階段階梯

⬜
1. 遙控車未開始
車能動、用鍵盤/手機能開。先搞定輪子、馬達、馬達驅動板。（純硬體，還沒 AI）
⬜
2. 眼睛接上未開始
鏡頭畫面進電腦，YOLO 能框出「草 / 障礙物」。（單獨玩 YOLO，先不上車）
⬜
3. 接大腦未開始
把模擬器調好的決策演算法，改成吃「YOLO 看到的東西」，自動開車。
⬜
4. 進階（很久以後）未開始
裝刀片、防水、電量 / 回充座。
